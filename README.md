# WolfCodes

This repository contains the source for **https://wolfcodes.dev**, an educational website focused on clear, conceptual explanations of programming and software fundamentals.

The site is authored and maintained by **Aaron Wolf**.

---

## About

WolfCodes is a documentation-first project covering:

- **Python**
  - Language reference
  - Beginner-friendly user guide
  - Advanced topics (object model, iteration, typing, concurrency, metaprogramming)

- **JavaScript**
  - Core language reference
  - Modern language features and APIs

- **Supporting topics**
  - Git and GitHub
  - Command line interfaces (CLI)
  - SQL and databases
  - System and tooling fundamentals

The emphasis is on **how things work**, not just how to use them.
